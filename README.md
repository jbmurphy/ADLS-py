# ADLS-py

- ADLS.py is a python script that will :
  - upload files to Azure Data Lake Storage (Gen 2)
  - create an PBIDS file pointing to the same ADLS

- ADLS.pbids is an example PBIDS for Azure Data Lake Storage (Gen 2)
